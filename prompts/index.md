# 提示词目录

这里的提示词可以复制给任何 AI（ChatGPT、Claude 等），让它帮你操作 rancher-ops 脚本。

## 使用方式

1. 打开对应的 `.md` 文件
2. 把 `[填...]` 替换成实际值
3. 整段复制粘贴给 AI

## 场景列表

| 文件 | 场景 |
|---|---|
| [migrate-rbac.md](./migrate-rbac.md) | 跨集群迁移 RBAC 权限 |
| [copy-structure.md](./copy-structure.md) | 拷贝项目/Namespace 结构 |
| [audit-rbac.md](./audit-rbac.md) | 审计 RBAC 权限配置 |
| [full-migration.md](./full-migration.md) | 完整迁移（结构 + 权限） |
