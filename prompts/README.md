# prompts — AI 提示词模板

让 AI 执行运维操作时用的提示词模板，复制粘贴即可。

## 模板列表

| 模板 | 用途 |
|------|------|
| `full-migration.md` | 完整迁移（跨 Rancher SSO / 同 Rancher 跨集群） |
| `healthcheck.md` | 执行集群巡检 |
| `migrate-rbac.md` | 仅迁移 RBAC 权限 |
| `copy-structure.md` | 仅复制项目/NS 结构 |
| `audit-rbac.md` | RBAC 审计/报告 |

## 使用

1. 复制对应模板内容
2. 填入参数（集群名、Rancher URL 等）
3. 粘贴给 AI 执行

## 关联

所有操作脚本在 https://github.com/kodyssss/rancher-ops。
